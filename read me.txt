This OSC script is made for VRChat.
You can edit the script itself or edit the config.json in the folder "files"

If you are new to nodeJS: please just edit the json



HOW TO RUN FILE:
 1.
	Run "install node.bat"
 2.
	open the folder in terminal
 3.
	type "npm install node-osc node-fetch"
 4.
	Run run.bat and it should be running
